OBJECTIVE

INVESTIGATE STATION SUBSYSTEMS. CONFIRM INFECTION. GATHER CORROBORATING EVIDENCE. THEN EXECUTE CONTAINMENT PROTOCOL.

THE SIMULATION ENDS ONLY WHEN CONTAINMENT IS EXECUTED.

RIGOR (DIFFICULTY)

RIGOR DETERMINES HOW MANY STRONG EVIDENCE SIGNALS YOU NEED FOR THE BEST OUTCOME. THE CONSOLE SHOWS THIS AS STRONG EVIDENCE REQUIRED: X/3.

  • RELAXED — 1/3 REQUIRED
  • STANDARD (DEFAULT) — 2/3 REQUIRED
  • STRICT — 3/3 REQUIRED

METERS

CAUTION
PROCEDURE / VERIFICATION. HIGHER CAUTION SUPPORTS RELIABLE DECISIONS.
PARANOIA
INSTABILITY / DISTRUST. SOME OPTIONS DISAPPEAR AS PARANOIA RISES.
RISK
TIME PRESSURE + SPREAD POTENTIAL. INCREASES EVERY 5 MINUTES, AND MAY SPIKE AFTER RISKY ACTIONS. AT RISK ≥ 3, A SECONDARY INFECTION MAY OCCUR ONCE (MAX 2 INFECTED TOTAL).

QUICK TOUR RUN (FAST DEMO)

  1. OPEN STATION CONSOLE (SHOW METERS + RIGOR HEADER)
  2. COLLECT STRONG EVIDENCE
    ON STANDARD, YOU WANT ANY TWO. ON STRICT, ALL THREE.
    • KENNEL → RUN PROVOCATION TEST → LOG ANOMALY
    • SECURITY PLAYBACK → LAST 4 HOURS → FLAG CONTRADICTION
    • GENERATOR → VIEW ROUTING TABLE → LOG UNAUTHORIZED ROUTE
  3. CONFIRM INFECTION
    • LAB → MANUAL BLOOD TESTS
    • VIOLENT REACTION = INFECTED (CONFIRMED)
    • NO REACTION = HUMAN
    BULK SCAN IS A SHORTCUT WITH CONSEQUENCES (RISK + UNCERTAINTY).
  4. KEEP RISK LOW (BEST OUTCOME TARGET: RISK ≤ 2 AT EXECUTION)
  5. FINAL ASSESSMENT → EXECUTE CONTAINMENT PROTOCOL

OPTIONAL / FLAVOR

  • CHESS WIZARD — MOVIE-ACCURATE MOVE SEQUENCE + CHECKMATE QUOTE + OPTIONAL WHISKEY SHUTDOWN.

ENDINGS (2 TOTAL)

CLEARANCE GRANTED (BEST)

  • INFECTION CONFIRMED (MANUAL BLOOD TEST RETURNS INFECTED)
  • STRONG EVIDENCE THRESHOLD MET (BASED ON RIGOR)
  • RISK ≤ 2 AT EXECUTION

REWARD: CLEARANCE GRANTED TROPHY DOWNLOAD.

BREACH RECORDED

TRIGGERS IF ANY SUCCESS CONDITION IS NOT MET: NO CONFIRMED INFECTION, INSUFFICIENT STRONG EVIDENCE FOR THE SELECTED RIGOR, OR RISK TOO HIGH AT EXECUTION.

REWARD: BREACH RECORDED TROPHY DOWNLOAD.

WHAT COUNTS AS “STRONG EVIDENCE” (THE 3 FLAGS)
  1. KENNEL — PROVOCATION TEST CONFIRMS ANOMALOUS RESPONSE
  2. SECURITY PLAYBACK — TIMELINE CONTRADICTION (ONE PERSON IN MULTIPLE PLACES)
  3. GENERATOR — UNAUTHORIZED ROUTE FEEDING HAB-03 (INVALID SIGNATURE)

THESE ARE THE ONLY FLAGS THAT COUNT TOWARD THE RIGOR THRESHOLD. OTHER ANOMALIES ADD FLAVOR / PRESSURE.

LAB DETAILS (BLOOD TESTS, BULK SCAN, SIMULATION)
  • MANUAL BLOOD TEST IS THE ONLY WAY TO CONFIRM INFECTION.
  • BULK SCAN IS FAST BUT RISKY: IT INCREASES RISK AND CAN MAKE READOUTS FEEL LESS TRUSTWORTHY, ENCOURAGING RETESTING.
  • ASSIMILATION SIMULATION SUMMARIZES CURRENT DATA. IF RUN TOO EARLY, IT DISPLAYS “DATA INSUFFICIENT” / “LIMITED DATA” WARNINGS.
INFECTION / SPREAD RULES
  • Every run starts with exactly 1 infected crew member (random identity).
  • At risk ≥ 3, the system may trigger one secondary infection (max 2 total).
  • This is meant to punish delays / chaos, not to create unwindable states.
CHESS WIZARD

THE CHESS PROGRAM MIRRORS THE FILM SEQUENCE:

  • Player: bishop to knight 4 (assertive) or king to rook 1 (withdrawal)
  • Computer responds, then declares: checkmate. checkmate.
  • Flavor line appears: “you cheating bitch.”
  • Optional: Pour whiskey to take chess wizard offline.
SHACK
  • Only appears if PARANOIA ≤ 2, no confirmed infection, and not already used.
  • Modifies meters (risk up, caution down), then returns to console.
  • Does not affect ending rules directly (it’s a human moment).
