WHAT THIS IS

A SHORT, REPLAYABLE TEXT SIMULATION INSPIRED BY THE THING. YOUR JOB IS TO GATHER EVIDENCE, CONFIRM INFECTION, THEN EXECUTE CONTAINMENT. THE SIMULATION ENDS ONLY WHEN CONTAINMENT IS EXECUTED.

DIFFICULTY / RIGOR (AFFECTS CLEARANCE)

THE SIM TRACKS 3 STRONG EVIDENCE SIGNALS. RIGOR DETERMINES HOW MANY ARE REQUIRED FOR CLEARANCE GRANTED.

  1. KENNEL — PROVOCATION ANOMALY
  2. SECURITY PLAYBACK — TIMELINE CONTRADICTION
  3. GENERATOR — UNAUTHORIZED ROUTE (HAB-03)
  • RELAXED — 1/3 REQUIRED
  • STANDARD (DEFAULT) — 2/3 REQUIRED
  • STRICT — 3/3 REQUIRED

THE CONSOLE DISPLAYS THIS AS: STRONG EVIDENCE REQUIRED: X/3.

METERS

CAUTION
PROCEDURE / VERIFICATION. HIGHER CAUTION SUPPORTS RELIABLE READOUTS AND CONTROL.
PARANOIA
INSTABILITY / DISTRUST. SOME OPTIONS DISAPPEAR AS PARANOIA RISES.
RISK
TIME PRESSURE + SPREAD POTENTIAL. RISK INCREASES AUTOMATICALLY EVERY 5 MINUTES, AND MAY SPIKE AFTER RISKY ACTIONS.

NOTE: AT RISK ≥ 3, A SECONDARY INFECTION MAY OCCUR ONCE (MAX 2 INFECTED TOTAL).

INFECTION MODEL

  • EACH RUN BEGINS WITH EXACTLY 1 INFECTED CREW MEMBER (RANDOM IDENTITY).
  • CONFIRMATION IS ACHIEVED VIA MANUAL BLOOD TESTING.

RECOMMENDED TOUR RUN (FAST DEMO)

  1. OPEN STATION CONSOLE
    THIS IS THE MAIN HUB. YOU CAN SHOW RIGOR + METERS HERE.
  2. GET STRONG EVIDENCE (ON STANDARD: PICK ANY TWO)
    • KENNEL → RUN BEHAVIORAL PROVOCATION TEST → LOG ANOMALY
    • SECURITY PLAYBACK → LAST 4 HOURS → FLAG TIMELINE CONTRADICTION
    • GENERATOR → VIEW ROUTING TABLE → LOG UNAUTHORIZED ROUTE (HAB-03)
    ON STRICT, YOU’LL WANT ALL THREE.
  3. CONFIRM INFECTION
    • LAB → CONDUCT MANUAL BLOOD TESTS
    • VIOLENT REACTION = INFECTED (CONFIRMED)
    • NO REACTION = HUMAN
    OPTIONAL: BULK ANOMALY SCAN IS A SHORTCUT WITH CONSEQUENCES (RISK + UNCERTAINTY).
  4. MANAGE RISK
    • FOR THE BEST OUTCOME, EXECUTE WITH RISK ≤ 2.
    • REMEMBER: RISK ALSO INCREASES AUTOMATICALLY EVERY 5 MINUTES.
  5. FINAL ASSESSMENT → EXECUTE CONTAINMENT PROTOCOL
    THIS IS THE ONLY ACTION THAT ENDS THE SIMULATION.

OPTIONAL CONTENT (NON-ESSENTIAL)

SHACK (EASTER EGG)

  • APPEARS ONLY IF: PARANOIA ≤ 2, NO CONFIRMED INFECTION, AND NOT ALREADY USED.
  • GRANTS AN OPTIONAL DOWNLOAD ON THAT SCREEN.
  • RETURNS TO CONSOLE (DOES NOT END THE SIMULATION).
  • MODIFIES METERS (RISK UP, CAUTION DOWN).

CHESS WIZARD (MOVIE-ACCURATE)

  • PLAYS OUT THE FILM’S MOVE SEQUENCE (ASSERTIVE VS WITHDRAWAL FRAMING).
  • INCLUDES THE CHECKMATE LINE + OPTION TO POUR WHISKEY ON THE COMPUTER.
  • WHISKEY DISABLES CHESS WIZARD AFTERWARD (THEMATIC MOMENT, NOT A WIN CONDITION).

ENDINGS (ONLY 2 — BOTH COMPLETE)

CLEARANCE GRANTED

  • INFECTION CONFIRMED (MANUAL BLOOD TEST RETURNS INFECTED)
  • STRONG EVIDENCE THRESHOLD MET (BASED ON RIGOR)
  • RISK ≤ 2 AT EXECUTION

REWARD: CLEARANCE GRANTED TROPHY DOWNLOAD.

BREACH RECORDED

TRIGGERS IF ANY SUCCESS CONDITION IS NOT MET (NO CONFIRMATION, INSUFFICIENT STRONG EVIDENCE FOR RIGOR, OR RISK TOO HIGH AT EXECUTION).

REWARD: BREACH RECORDED TROPHY DOWNLOAD.
